# GEC - Gerenciamento de Estoque e Controle
  Um aplicação web que facilita o gerenciamento de estoque de micro estabelecimentos (justamente pensado para isso).

  Armazenaamento de informações em um banco de Dados Sqlite.
  
  Aplicação criada a partir da linguagem Python, com seu framework web: Flask, e o uso de templates html. Com uso de BootStrap para CSS.
